## HTML: Block vs Inline elements

> This property describes how an element fits in with the flow of a document

---

### Block elements

- takes up the entire width of it's parents container.
- Always start on a new line and force subsequent elements onto a new line.
- header elements: `<h1>, <h2>, <p>`
- `<div>` content division element. It is a generic element used to group elements together. So in terms of web application layout, we can use the `<div>` element to group related elements together as part of laying out our website, where we can style these elements using CSS, or other CSS frameworks. You can call it a **block level container**.

---

### Inline Elements

- Anchor tag: `<a>` | `<span>
- `<span>`: this is also a generic element, however it is an inline element. So you can call it an inline container
